# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of SkillSync seriously. If you discover a vulnerability, we kindly ask you to report it immediately. 

**Do not open a public issue.** 

Instead, please send an email to the security team with a description of the vulnerability and steps to reproduce it. We will aim to acknowledge the report within 48 hours and provide a timeline for mitigation.
